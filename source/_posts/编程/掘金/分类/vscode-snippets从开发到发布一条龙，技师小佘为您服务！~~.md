<h2 data-id="heading-5">怎么写模版片段呢？</h2>
<p>        其实就是跟我们原来写代码一样，把我们想快捷生成的代码丢上去即可，以及有一些高级用法比如设置光标位置、占位符、变量等等<a href="https://link.juejin.cn/?target=https%3A%2F%2Fcode.visualstudio.com%2Fdocs%2Feditor%2Fuserdefinedsnippets%23_snippet-syntax" target="_blank" rel="nofollow noopener noreferrer" title="https://code.visualstudio.com/docs/editor/userdefinedsnippets#_snippet-syntax" ref="nofollow noopener noreferrer">片段语法传送门</a>  。</p>
<p>        <strong>现在我们知道了怎么写，怎么在本地跑起来，那么我们要怎么才能发布到vscode市场让小伙伴们一起玩呢?</strong></p>
<h2 data-id="heading-6">如何发布将自己的片段发到vscode市场呢</h2>
<h3 data-id="heading-7">1.全局安装依赖</h3>
<h4 data-id="heading-8">vsce 包管理工具</h4>
<pre><code class="copyable">npm install -g vsce
<span class="copy-code-btn">复制代码</span></code></pre>
<h4 data-id="heading-9">项目脚手架</h4>
<pre><code class="copyable">npm install -g yo generator-code
<span class="copy-code-btn">复制代码</span></code></pre>
<h3 data-id="heading-10">2.快速创建项目</h3>
<pre><code class="copyable">yo code

<p><strong>选择new code snippets</strong></p>
<p><img src="https://p1-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/36a5f690dac34b39bbb9b6b5c644b385~tplv-k3u1fbpfcp-watermark.image" alt="image.png" loading="lazy" referrerpolicy="no-referrer"></p>

<h3 data-id="heading-13">5.开发snippets且发布</h3>
<pre><code class="copyable">vsce publish    //拓展发布
vsce publish 2.0.1  //拓展发布至指定版本
vsce publish minor   //自动增加扩展版本
vsce unpublish (publisher name).(extension name) //取消发布扩展
<span class="copy-code-btn">复制代码</span></code></pre>
<h3 data-id="heading-14">6.敲重点的一节来了package.json属性含义讲解</h3>
<ol start="2">
<li>package.json 文件图解</li>
</ol>
<ul>
<li>version(红色区域)：版本号每次有修改且需要发布时一定要修改版本号不然发不上去（此坑踩过）</li>
<li>engines(蓝色区域)：最低vscode版本号支持，就是vscode1.28.0版本以上的才能使用此拓展</li>
<li>repository(黄色区域): 远程仓库地址</li>
<li>contributes(白色区域): snippets配置项可设置language、以及aliases设置别名等等</li>
</ul>
