<h1 data-id="heading-0">使用<a href="https://link.juejin.cn/?target=https%3A%2F%2Fvuepress-theme-reco.recoluan.com%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://vuepress-theme-reco.recoluan.com/" ref="nofollow noopener noreferrer">vuepress-theme-reco</a>搭建自己的博客</h1>
<p>之前在年中总结<a href="https://juejin.cn/post/6975081827916578853" target="_blank" title="https://juejin.cn/post/6975081827916578853">陌上花开浅浅行｜2021 年中总结</a>提到过我的博客，大家可以去看看，下面是搭建方法。</p>
<p><a href="https://link.juejin.cn/?target=https%3A%2F%2Fvuepress-theme-reco.recoluan.com%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://vuepress-theme-reco.recoluan.com/" ref="nofollow noopener noreferrer">vuepress-theme-reco</a>是VuePress的一个主题，大家可以参考VuePress和其他主题，搭建你自己风格的主题，这篇主要讲解<a href="https://link.juejin.cn/?target=https%3A%2F%2Fvuepress-theme-reco.recoluan.com%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://vuepress-theme-reco.recoluan.com/" ref="nofollow noopener noreferrer">vuepress-theme-reco</a>主题的博客。</p>
<p>VuePress是静态网页生成器，通过它，可以迅速的将我们的Markdown笔记变成网页，方便我们翻阅，查找。</p>
<h2 data-id="heading-1">一、初始化项目</h2>
<h3 data-id="heading-2">1、引入<a href="https://link.juejin.cn/?target=https%3A%2F%2Fvuepress-theme-reco.recoluan.com%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://vuepress-theme-reco.recoluan.com/" ref="nofollow noopener noreferrer">vuepress-theme-reco</a>主题</h3>
<pre><code class="hljs language-bash copyable" lang="bash">npm install vuepress-theme-reco --save-dev
<span class="copy-code-btn">复制代码</span></code></pre>
<h3 data-id="heading-3">2、使用官方的模板初始化项目</h3>
<pre><code class="hljs language-bash copyable" lang="bash">theme-cli init blogs
<span class="copy-code-btn">复制代码</span></code></pre>
<p>按要求输入内容</p>
<p><img src="https://p9-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/161155dd4c6748898f2ba1d8cb32eba1~tplv-k3u1fbpfcp-watermark.image" alt="image-20210712152501859.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>将在你的文件夹下生成你项目名的文件夹，我这里是blog.</p>
<h3 data-id="heading-4">3、运行</h3>
<p>进到文件夹内，使用cmd执行</p>
<pre><code class="hljs language-bash copyable" lang="bash">npm install
npm run dev
<span class="copy-code-btn">复制代码</span></code></pre>
<p><img src="https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/552776085cd64b53bf785a85600a679d~tplv-k3u1fbpfcp-watermark.image" alt="image-20210712133159335.png" loading="lazy" referrerpolicy="no-referrer"></p>
<p>至此博客已经可以正常使用了，但是我们打开官方的案例，往往比我们现在的效果要好，我们需要优化的继续往下看。</p>
<h2 data-id="heading-5">二、优化页面</h2>
<h3 data-id="heading-6">1、官方的功能</h3>
<p>像评论、时间线、域名的配置、多语言、暗色适配、侧边栏、文档的摘要...官方写的已经很详细，这里就不再介绍，按照<a href="https://link.juejin.cn/?target=https%3A%2F%2Fvuepress-theme-reco.recoluan.com%2F" target="_blank" rel="nofollow noopener noreferrer" title="https://vuepress-theme-reco.recoluan.com/" ref="nofollow noopener noreferrer">官方文档</a>配置即可。如果有疑问，可以在下面评论，咨询我。</p>
