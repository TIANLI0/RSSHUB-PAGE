<p data-vmark="4305"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 18 日消息，据IT之家网友 @Rongronggg9 投稿，微软今日为 Dev 渠道的用户推送了 <span class="accentTextColor"><a class="s_tag" href="https://win11.ithome.com/" target="_blank">Win11</a> Build 22504 预览版</span>。</p><h2 data-vmark="605b">本次更新的亮点有：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="a795">触控键盘的 13 个主题扩展到其他输入体验，包括输入法、表情符号面板和语音输入。新版还增加了根据家庭成员、心有灵犀的情侣、亲吻和手牵手的人的面部和肤色进行个性化表情符号组合的功能。</p></li><li><p data-vmark="ef19"><span class="accentTextColor">你的手机 App 推出全新设计</span>，面向 Dev 渠道用户测试。</p></li><li><p data-vmark="83e9">由于美国感恩节假期，团队的许多成员都计划休假一周，因此<span class="accentTextColor">下周不会有 Win11 Dev 预览版发布</span>。</p></li></ul><h2 data-vmark="3123">新增内容：</h2><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="8029">为了进一步个性化 Windows 文本输入体验，<span class="accentTextColor">微软扩展了触摸键盘的 13 个主题，现在适用于其他输入体验，包括键盘输入法、表情符号面板和语音输入</span>。此外，主题引擎允许用户创建完全自定义的主题，包括背景图像。用户可以在“设置”>“个性化”下的“文本输入”部分下找到这些设置。</p></li></ul><p data-vmark="eb83"><img src="https://img.ithome.com/newsuploadfiles/2021/11/9a97f98e-b9c8-4096-9176-f2570b3ba11f.png" w="1440" h="792" title="微软 Win11 Dev 预览版 Build 22504 发布：输入法迎来自定义主题，解决键盘灯不亮问题" width="1440" height="451" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="cc7a">增加了根据家庭成员、心有灵犀的情侣、亲吻和手牵手的人的面部和肤色进行个性化表情符号组合的功能。只需打开表情符号面板 (WIN + .) 并通过在搜索框中键入以下内容来尝试这些组合：家庭、情侣、牵手或接吻。</p></li></ul><p data-vmark="ed08"><img src="https://img.ithome.com/newsuploadfiles/2021/11/a9a6dff8-fcbb-4aa2-bceb-7381c92a5475.png" w="1440" h="757" title="微软 Win11 Dev 预览版 Build 22504 发布：输入法迎来自定义主题，解决键盘灯不亮问题" width="1440" height="431" referrerpolicy="no-referrer"></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="bddd">现在可以使用 WIN + Alt + K 键盘快捷键切换任务栏中的麦克风静音状态。</p></li></ul><p data-vmark="6a9f"><img src="https://img.ithome.com/newsuploadfiles/2021/11/d9ceb6e6-4d2c-4666-838d-d2eb6d64f34d.png" w="840" h="280" title="微软 Win11 Dev 预览版 Build 22504 发布：输入法迎来自定义主题，解决键盘灯不亮问题" width="840" height="273" referrerpolicy="no-referrer"></p><ul class=" list-paddingleft-2"><li><p data-vmark="50ed">根据反馈，微软正在使用包含变音符号的关键字使葡萄牙语和波兰语的表情符号搜索更加灵活。</p></li><li><p data-vmark="dec2"><span class="accentTextColor">.NET Framework 4.8.1 上线，带来原生的 ARM64 支持</span>。</p></li><li><p data-vmark="e217">由于 Win11 中的“设置”>“系统”>“声音”会在用户的隐私设置中限制麦克风访问时显示警告，因此微软将删除通知以提醒用户注意这一点。</p></li><li><p data-vmark="3d0a">添加了对取消斐济共和国 2021 年夏令时的支持。</p></li><li><p data-vmark="d171">新安装 Win11 时，触摸指示器选项现在默认关闭，位于“设置”>“辅助功能”>“鼠标指针和触摸”下。从“设置”>“蓝牙和设备”>“触摸”添加了一个指向此设置页面的链接。</p></li><li><p data-vmark="3fc7"><span class="accentTextColor">当在小型设备（11 英寸屏幕对角线及以下）上处于平板电脑模式时，应用程序将默认最大化启动</span>。</p></li></ul><h2 data-vmark="635a">修复：</h2><p data-vmark="3009"><strong>开始菜单：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="592f">修复了开始菜单推荐部分中图标显示不正确的一个根本原因。</p></li><li><p data-vmark="8398">如果用户打开开始菜单并立即按 Shift + F10 或上下文菜单键，则上下文菜单现在应该与搜索框对齐。</p></li></ul><h2 data-vmark="ef30">任务栏：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="a7db">修复了在任务栏中加载电池图标时出现问题，导致在最近的版本中 explorer.exe 崩溃的问题。</p></li><li><p data-vmark="2d65">进行了一些更改以帮助任务栏中的时钟更可靠地更新，以及在远程桌面上时钟不更新的问题。</p></li><li><p data-vmark="8d7b">缓解了可能导致日期和时间从任务栏末端被裁剪掉的定位问题。</p></li><li><p data-vmark="5384">如果用户自定义了低电池电量通知，任务栏中的电池图标将能够正确显示，而不是使用默认值来显示警告。</p></li><li><p data-vmark="a8b2">右键点击任务栏中的日期和时间现在将关闭打开的通知中心，以便用户可以看到上下文菜单选项。</p></li></ul><h2 data-vmark="3ff7">输入：</h2><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="d2be"><span class="accentTextColor">键盘灯（例如大写锁定）现在应该可以正常工作了</span>。</p></li><li><p data-vmark="ee97">根据反馈对表情符号搜索关键字进行了更多调整，包括改进白天、黑夜的搜索结果。</p></li><li><p data-vmark="3840">修复了当使用中文输入法进行 WIN + . 时，颜文字 kaomoji 列表空白的问题。</p></li><li><p data-vmark="3d78">修复了表情符号面板搜索结果的问题，该问题可能导致单个 gif 与某些查询重复显示。</p></li><li><p data-vmark="6ecc"><span class="accentTextColor">缓解了输入法不适用于某些游戏的问题</span>。</p></li><li><p data-vmark="3826">解决了“设置”>“个性化”下更改触摸键盘设置“显示文本建议并在选择文本建议后添加空格键”下的选项可能意外不显示的问题。</p></li><li><p data-vmark="1c2c">如果用户在使用语音输入 (WIN + H) 时暂停很长时间，现在应该能够恢复正常使用。</p></li><li><p data-vmark="9ee7">修复了语音输入中实际聆听状态和麦克风视觉之间的一些不一致。</p></li><li><p data-vmark="b290">如果用户在使用语音输入时移动插入符号，文本不应再意外地重复。</p></li></ul><p data-vmark="514f"><strong>窗口：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="fe03"><span class="accentTextColor">选择对齐布局选项不再随机地将窗口放在其他显示器上</span>。</p></li></ul><p data-vmark="159b"><strong>设置：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="541a"><strong>快速设置中的亮度和音量滑块已可正确显示</strong>。</p></li><li><p data-vmark="6179">修复了无法手动设置调整活动时间的问题。</p></li><li><p data-vmark="ad3d">解决了在快速设置中连接到网络时可能导致触摸键盘遮挡密码输入字段的问题。</p></li></ul><h2 data-vmark="0c83">其他：</h2><ul class=" list-paddingleft-2"><li><p data-vmark="0702">修复了导致更新提示对话框（以及该样式的其他对话框）在 DPI 更改后被截断且无法正确绘制的缩放问题。</p></li><li><p data-vmark="bfb3">改进了右键点击桌面底角时上下文菜单的定位。</p></li><li><p data-vmark="8074">缓解了与 TCPIP 相关的问题，该问题可能导致在唤醒设备时进行错误检查。</p></li><li><p data-vmark="4588">修复了高优先级通知（例如警报）有时会在用户不参与的情况下意外关闭的问题，导致它在下次正常优先级通知进来时随机重新出现。</p></li></ul><h2 data-vmark="d764">已知问题：</h2><p data-vmark="545b"><strong>常规：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="3811">使用最新的 ISO 从版本 22000.xxx 或更早版本更新到更新的开发人员渠道的用户可能会收到错误警告；</p></li><li><p data-vmark="e23e">正在调查某些 PC 无法安装新版本或其他更新的问题，PC 可能会报告错误代码 0x80070002。如果你遇到此问题，请重新启动 PC 并重试。</p></li><li><p data-vmark="fd78">安装此版本时，某些 PC 可能会回滚并显示错误代码 0xc1900101-0x4001c。如果你点击了回滚选项，在 PC 回滚到之前的版本后，可能需要暂停更新，直到微软发布修复程序。</p></li></ul><p data-vmark="2aa6"><strong>开始菜单：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="08a4">在某些情况下，使用“从开始搜索”或任务栏时可能无法输入文本。如果遇到此问题，请按键盘上的 WIN+R 以启动“运行”对话框，然后将其关闭。</p></li></ul><p data-vmark="0a84"><strong>任务栏：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="0bf7">切换输入法时，任务栏有时会闪烁。</p></li></ul><p data-vmark="5324"><strong>窗口：</strong></p><ul class=" list-paddingleft-2"><li><p data-vmark="da8f">在任务视图中的不同桌面之间来回悬停鼠标将导致显示的缩略图和内容区域意外缩小。</p></li></ul><p data-vmark="3d5b"><strong>搜索：</strong></p><ul class="ai-word-checked list-paddingleft-2"><li><p data-vmark="e926">点击任务栏上的搜索图标后，搜索面板可能无法打开。如果发生这种情况，请重新启动“Windows 资源管理器”进程，然后再次打开搜索面板。</p></li></ul>
