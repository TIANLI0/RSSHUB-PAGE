级驱逐舰的改进潜力又已经被彻底榨干，也无法承担未来与中国海军竞争的主力角色。</p><p>建造中的最新批次 " 阿利 · 伯克 " 级</p><p>无可奈何的美国海军只能将 " 朱姆沃尔特 " 级的部分成熟技术与 " 阿利 · 伯克 " 级的最新武器系统结合起来，推出性能在前两者之间的 DDG ( X ) ，希望它能在性能与成本、前瞻与务实上取得一个相对平衡的结果。</p><p>而这种平衡的结果，就是现在外界看到的这个被嘲讽为 "055 山寨品 " 的设计。</p><p>根据美国海军学会网站的报道，DDG ( X ) 是美国海军 20 多年来尝试建造的最大吨位水面作战舰艇之一，它的排水量高达 1.2 万吨，不再采用 " 朱姆沃尔特 " 级驱逐舰那样的科幻外形，转而回归传统设计。它它需要具备充沛的电能才能驱动新一代定向能武器和高功率传感器。DDG ( X ) 将使用 " 阿利 · 伯克 " 级驱逐舰配备的最新作战系统，结合了全新研制的 SPY-6 有源相控阵雷达和 " 基线 10" 版本宙斯盾系统。为提升探测灵敏度，SPY-6 雷达增大了天线阵列面积，但需要的电能也随之倍增。此外，该舰还计划安装功率高达 600 千瓦的激光武器以拦截敌方导弹。为此，DDG ( X ) 将移植 " 朱姆沃尔特 " 级驱逐舰使用的综合动力系统，利用燃气轮机带动大功率发电机以满足全舰对电力的需要。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img
