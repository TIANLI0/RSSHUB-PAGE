【Gemini任命AWS前高管为首席技术官】金色财经报道，加密交易平台宣布Pravjit从Amazon Web Services (AWS) 加入，此前他担任边缘服务总经理，包括 Amazon CloudFront、AWS 边缘计算和 AWS 数据传输操作。在 AWS 工作 6 年以上之前，Pravjit 在微软和迪士尼担任产品和工程领导职务。作为Gemini的首席技术官 (CTO)，Pravjit 将领导公司的工程和产品团队。Pravjit表示决定加入Gemini是因为“我想帮助构建 Web3 和去中心化的元宇宙”。（finextra）
