【Mudrex推出基于API的企业级“加密即服务”解决方案】12月21日消息，加密货币资产管理公司Mudrex推出了一种基于API的“加密即服务”解决方案，旨在使传统经纪商和加密经纪商、投资组合管理服务和金融科技公司都能提供安全的加密投资产品。 
据悉，该方案为经纪公司提供KYC、监管和合规监控。同时，加密货币的托管和处理将由Mudrex管理。技术框架完全由全球平台构建和支持。 
该公司补充说，使用这项服务，客户将能够为他们的公司建立一个单一的中央钱包，或者为他们的每个用户建立无数个钱包。在推出时，Mudrex将支持70个不同的国家。（Economic Times）
