<a href="https://pyfound.blogspot.com/2022/03/meta-deepens-its-investment-in-python.html" target="_blank"><strong>Meta 公司近日向 Python 软件基金会（PSF）提供了 30 万美元。</strong></a>该组织负责维护 Core Python（CPython），这是一项为大多数机器学习（ML）和人工智能应用提供动力的开源编程语言。<br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0325/08127b5aaa99a76.webp" alt="u33ccso4.webp" referrerpolicy="no-referrer"></p><p style="text-align: center;">图片来自于 <a href="https://commons.wikimedia.org/wiki/File:Python_image.jpg" target="_blank">Wikimedia</a></p><p style="text-align: left;">PSF 表示：“Python 对于 Meta 来说非常重要。PyTorch 是建立在 Python 基础上的，加速了 Meta 内部以及整个开源 ML 生态系统，加速推进从 ML 研究和原型设计到生产的路径。Cinder 是 Meta 面向性能的 Python 版本，允许 Instagram 在全球范围内运行，而 Pyre 是 Meta 成千上万的 Python 开发人员使用的高性能类型检查器”。</p><p style="text-align: left;">PSF 在一篇博文中说：“Meta 对 PSF 的投资将为 PSF 提供关键支持，并为成功的驻场开发者计划提供第二年的资金”。</p><p style="text-align: left;">PSF 的“常驻开发者”（Developer-in-Residence）计划于 2021 年启动，为 CPython 的全职开发者角色提供资金。该计划帮助 PSF 雇佣了 Łukasz Langa，他一直在“处理积压的拉动请求，完成bugs.python.org向GitHub问题的迁移，以及指导新的核心开发人员”。</p>
