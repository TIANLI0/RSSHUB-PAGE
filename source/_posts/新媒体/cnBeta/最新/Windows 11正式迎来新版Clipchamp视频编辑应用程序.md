如果你一直在等待 Movie Maker 视频编辑应用程序重返 Windows 操作系统，那 Clipchamp 或许是一个更好的选择。<strong>在去年收购了该公司后，微软现正在将这款软件捆绑到 Windows 11 之中。</strong>可知作为一款基于浏览器的应用程序，Clipchamp 支持修剪、拆分、过渡，甚至屏幕录像等功能。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0310/61272f753113b0e.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0310/61272f753113b0e.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">截图（来自：Microsoft Store）</p><p>与 Adobe Premiere Pro 等专业级视频编辑应用程序一样，Clipchamp 也包括了一条时间线。可使用模板、滤镜、过渡、以及媒体样式库等组合，界面设计易于上手，可轻松创建并输出具有多条音轨的内容。</p><p><img src="https://static.cnbetacdn.com/article/2022/0310/a6c6d922f6f18c5.gif" alt="2.gif" referrerpolicy="no-referrer"></p><p style="text-align: center;">时间轴（via Windows <a href="https://blogs.windows.com/windows-insider/2022/03/09/announcing-windows-11-insider-preview-build-22572/" target="_self">Blogs</a>）</p><p>有趣的是，微软为 Clipchamp 集成了一套基于 Azure 云服务的文本转语音（TTS）生成器，以便创作者使用 70 种语言的旁白，此外还可 OneDrive 云存储联动。</p><p><img src="https://static.cnbetacdn.com/article/2022/0310/fb2cfe1f812b325.png" alt="1.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">功能一览</p><p>微软表示，Clipchamp 现已成为 Windows 11 预装应用的一员，此外该公司正在带来搜索体验方面的测试改进。</p><p><img src="https://static.cnbetacdn.com/article/2022/0310/d5b6a34c0f8725c.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">新搜索亮点功能</p><p>Windows Insider 项目负责人 Amanda Langowski 解释称：下周起的更新，将为“开始”与搜索框带来定期更新的内容（比如有趣的插曲），以帮助用户发现更多、保持联系、以及高效工作。</p><p><img src="https://static.cnbetacdn.com/article/2022/0310/c2561e90783cde0.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">工作用户可登录并访问公司文件</p><p>此外微软将持续改进并清除 Windows 上的一些顽疾，比如打印队列的更新会更匹配 Windows 11 的总体设计风格，Quick Assist 和 Windows Sandbox 迎来基于 Fluent 设计语言的图标。</p><p>最后，Windows Media Player 已被重命名为 Windows Media Player Legacy，意味着它将很快在 Windows 11 操作系统中彻底消失。如果一切顺利，上述改动将于未来几周内向所有用户推送。</p>
