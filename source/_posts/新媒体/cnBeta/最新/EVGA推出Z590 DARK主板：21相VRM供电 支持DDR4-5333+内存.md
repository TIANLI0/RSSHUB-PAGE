EVGA 终于为英特尔 11 代 Rocket Lake 台式 CPU 用户，推出了一款旗舰规格的 Z590 DARK 主板。<strong>可知由 KINGPIN 亲自设计的这款产品，不仅提供了多达 21 相的 VRM 供电、还支持 DDR4-5333+ 的内存。</strong>而且为了让发烧友们获得极致的超频体验，EVGA 还为 Z590 DARK 主板选用了不常见的水平双内存槽设计。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/0716/1b8319fb8ca8b5b.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0716/1b8319fb8ca8b5b.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：EVGA <a href="https://www.evga.com/products/product.aspx?pn=121-RL-E599-KR" target="_self">官网</a>）</p><p>据悉，Z590 芯片组兼容基于 LGA 1200 插槽的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a>第 10 / 11 代处理器，此外主板右侧的接口，都采用了 90° 直角的设计。</p><p>除了 24-pin 主板供电，EVGA Z590 DARK 主板还提供了双 8-pin 的 CPU 辅助供电连接器，不少理线强迫症用户应该都会喜欢这种侧置连接器方案。</p><p><a href="https://static.cnbetacdn.com/article/2021/0716/31ea2046411af01.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0716/31ea2046411af01.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>供电方面，Z590 DARK 采用夸张的 21 相数字 VRM 方案，并且覆盖了通向南桥（PCH）的硕大热管 + 散热片。</p><p>为提升美观度和高级感，EVGA 还为该主板提供了漂亮的金属框架覆盖件，辅以带有 RGB 灯的 I/O 挡板。</p><p><a href="https://static.cnbetacdn.com/article/2021/0716/b85e99fc5591814.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0716/b85e99fc5591814.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>虽然图片中看不到，但其实 I/O 挡板位置还内置了双 PWM <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C738%2C751" target="_blank">风扇</a>，用于增强供电模组的散热效果。</p><p>两条 DDR4 DIMM 插槽最高支持 5333+（OC）的内存频率，之所以选用了非传统的水平放置方式，则是为了让它更接近内存插槽，从而提供更好的信号、以及冲击更高的频率。</p><p><a href="https://static.cnbetacdn.com/article/2021/0716/13fb2a504d27b06.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0716/13fb2a504d27b06.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>虽然 EVGA Z590 DARK 最大只支持 64GB 的容量（32GB * 2 双通道），但这对于主流平台来说已经足够。</p><p>扩展方面，该主板提供了两条 PCIe 4.0 x16、一条 PCIe 3.0 x4、以及三个 M.2 插槽。其中一条 M.2 插槽支持 PCIe 4.0，且顶上被硕大的散热片给覆盖。</p><p><a href="https://static.cnbetacdn.com/article/2021/0716/ec8472c17500b54.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0716/ec8472c17500b54.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p><strong>以下是 EVGA Z590 DARK 主板的一些主要特性：</strong></p><blockquote><p>● 插槽金含量增加 150%</p><p>● 支持 PhysX 与 SLI 双卡交火</p><p>● 专为 EVGA ELEET X1 性能调教软件打造</p><p>● 提供双 EVGA Probe-It 连接器</p><p>● EVGA 最新的 GUI BIOS，自带 OC Robot 与 In-BIOS 压力测试。</p><p>● 背板带有 HDMI 2.0b 和 DisplayPort 1.4 视频输出端口</p><p>● 高效 21 相数字 VRM 供电模块</p><p>● 兼容英特尔傲腾（Optane）存储</p><p>● 集成英特尔双频 Wi-Fi 6 / 蓝牙 5.2 无线模块</p><p>● 提供 ARGB / RGB 接口，可通过 EVGA ELEET X1 进行控制。</p><p>● 板载 CMOS 清除、电源、以及重启键</p><p>● 板载温度和电压监测功能</p><p>● 支持 PCIe 4.0，提供禁用开关，加固 PCIe 插槽。</p><p>● 瑞昱（Realtek）7.1 高清音频 + EVGA NU Audio 支持</p><p>● 提供 SafeBoot 按钮，可重置主板并载入上次已知的正确 BIOS（无需清除 CMOS）。</p><p>● 调速开关选项，可将 CPU 倍频锁定到尽可能低的设置，允许用户实时切换最高 OC / 低速状态。</p><p>● 支持 USB 端口的 SPI 刷新，无需 CPU 即可刷新 BIOS 。</p><p>● 支持 Intel LGA 1200 插槽的 11 / 12 代处理器</p><p>● 三重 BIOS 支持</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2021/0716/4e4955947bc9b00.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0716/4e4955947bc9b00.png" alt="5.png" referrerpolicy="no-referrer"></a></p><p>存储方面，EVGA Z590 DARK 提供了 8 个 SATA 3 和一个 U.2 端口（均为 90° 横置），此外 PCB 上音频组件旁边有一个 6-pin 的 PCIe 辅助供电接口。</p><p>背板 I/O 方面，其包括了 7.1 声道音频插孔、4 个 USB 3.1 Gen 2 Type-A、1 个 USB 3.1 Gen 2 Type-C、2 个 USB 3.1 Gen 1、PS/2 键鼠复合端口、DP + HDMI 视频输出、以及双 2.5GbE 以太网 + 两根天线螺口。</p><p><a href="https://static.cnbetacdn.com/article/2021/0716/391016f6c9d832a.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0716/391016f6c9d832a.png" alt="6.png" referrerpolicy="no-referrer"></a></p><p>主板上的 BIOS 重置和电源按键旁边，还有双 Probelt 接头和多功能 POST 状态指示灯。</p><p>最后，EVGA Z590 DARK 主板的建议零售价为 599 美元（约 3873 RMB），且 DARK 主板系列还有 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> X570 芯片组的衍生版本。</p>
