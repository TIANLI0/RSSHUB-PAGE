<strong>微软一直在打磨 Windows 11 的体验。在今天发布的 Windows 11 Build 22567 预览版中，微软对“打开方式”（Open with）对话框进行了升级。</strong>总的来说，新的对话框看起来和感觉都更现代，这显然是正确的，尤其是 Windows 11 需要更多的打磨，以提供每个人所期望的精致体验。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0303/b3fe28e18f296f6.webp" target="_blank"><img src="https://static.cnbetacdn.com/article/2022/0303/b3fe28e18f296f6.webp" referrerpolicy="no-referrer"></a></p><p>值得注意的是，新界面目前并未向所有 Dev 频道的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> Insider 成员开放，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>表示该功能将会分阶段推出。微软表示：“我们已经更新了‘打开方式’对话框，以符合 Windows 11 的设计原则。更新后的对话框采用了浅色/深色主题。我们还简化了体验，只需一次点击就可以更新你的默认应用程序”。</p>
