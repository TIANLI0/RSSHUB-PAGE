奥睿科（ORICO）刚刚推出了两款存储 / 连接新品，<strong>其中包括 TB3-S2 便携式雷电扩展坞，以及配备了指纹生物识别传感器的安全型移动硬盘。</strong>首先是外形相当紧凑的 TB3-S2 雷电扩展坞，其长宽高为 18.8 × 9.5 × 2.4 厘米，且支持雷电 3（40 Gbps）菊链和 8K @ 60Hz 连接。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1228/6a04b353446eda2.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1228/6a04b353446eda2.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>奥睿科 TB3-S2 雷电扩展坞附带了 120W 电源适配器，支持基于雷电 3 接口的 DisplayPort 1.3 显示输出，内部还有两个 M.2-2280 NVMe 插槽（支持 4TB 容量）。</p><p><img src="https://static.cnbetacdn.com/article/2021/1228/c4947ed61819fe1.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>下行端口方面，TB3-S2 扩展坞提供了两个 10Gbps 的 USB 3.2 Gen 2（其中一个是 C 口）、两个 5Gbps 的 USB 3.2 Gen 1（Type-A 口）、一对 60W / 15W 的 USB-PD 充电口。</p><p><img src="https://static.cnbetacdn.com/article/2021/1228/6900c2ad3243929.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>另有千兆（1 GbE）以太网卡、以及支持麦克风输入的双通道音频插孔（通过 USB DAC 实现）。奥睿科 TB3-S2 雷电扩展坞现已上市，售价为 399.99 美元（约 2549 RMB）。</p><p><img src="https://static.cnbetacdn.com/article/2021/1228/e6142f117a7551a.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>然后是 ORICO P<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://list.jd.com/list.html?cat=670,677,11303" target="_blank">SSD</a>，作为一款移动硬盘，其可选 256GB / 512GB / 1TB 三档容量，且坚固的铝制机身能够为内置硬盘提供良好的散热。</p><p>该产品通过一根 10Gbps 的 USB 3.2 Gen 2 线缆与主机设备连接，支持 UASP 协议（基于 USB 的 SCSI 协议），支持搞到 520 MB/s 的传输速率。</p><p><img src="https://static.cnbetacdn.com/article/2021/1228/1064516883e6a4e.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>值得一提的是，ORICO PSSD 还支持基于指纹传感器的生物识别安全特性，辅以兼容 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 7+、macOS、Linux、Android 系统平台的专用解锁软件。</p><p>价格方面，256GB / 512GB / 1TB 版本分别售 79.99 / 99.99 / 159.99 美元（约合 510 / 637 / 1019 RMB）。</p>
