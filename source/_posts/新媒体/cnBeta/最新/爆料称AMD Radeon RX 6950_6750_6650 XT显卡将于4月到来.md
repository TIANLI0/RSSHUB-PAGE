知名爆料人 @热心市民描边怪，刚刚在芯域（Chiphell）论坛上分享了与 AMD“RNDA 2 Refresh GPU”有关的产品信息。<strong>可知新阵容中包括了 RX 6950 XT、RX 6750 XT 和 RX 6650 XT 等 SKU，且有望于下月正式发布。</strong>有趣的是，在这三款显卡之外，红队还有准备提供 RX 6500（非 XT 型号）。与主打 DIY 入门市场的  RX 6400 相比，RX 6500 主要面向 OEM 渠道。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0308/934fcfeb0a364ee.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0308/934fcfeb0a364ee.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://wccftech.com/amd-rdna-2-gpu-radeon-rx-6950-xt-6750-xt-6650-xt-graphics-cards-april-launch-rumor/" target="_self">WCCFTech</a>）</p><p><strong>@热心市民描边怪 指出：</strong></p><blockquote><p>4 月 20-21 日将有几款新款 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 新卡到来，其中 RX 6950XT、RX 6750XT、RX 6650XT 会比当前在售的 Radeon 6000（非 -50 型号）贵一点。</p><p>三款显卡都升级了 18Gbps 的 GDDR6 显存，因而性能也有小幅提升，此外有一款低端亮机卡 RX 6400 。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0308/5a3c7116bca4b75.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0308/5a3c7116bca4b75.png" alt="0.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">截图（来自：<a href="https://www.chiphell.com/forum.php?mod=viewthread&tid=2398696" target="_self">Chiphell</a>）</p><p>据悉，三款 Radeon RX 6X50 XT 显卡，将分别配备刷新后的 AMD Navi 21、Navi 22 和 Navi 33 GPU 。</p><blockquote><p>● Navi 21 GPU 对应当前在售的 Radeon RX 6900 XT、RX 6800 XT 和 RX 6800 显卡型号。</p><p>● Navi 22 GPU 有为 RX 6700 XT 提供支撑，而 Navi 23 GPU 被用于台式机上的 RX 6600 / 6600 XT 。</p></blockquote><p>显存方面，除了即将推出的主流 RX 6500 XT 和旗舰 RX 6900 XT LC 显卡之外，其它 SKU 都沿用 16Gbps 显存芯片。</p><p>通过升级 18Gbps GDDR6 显存，预计可带来 2~5% 的性能提升。此外我们听说 RX 6950 XT 具有超过 2.5 GHz 的 GPU 时钟频率，并将获益于增强的芯片制造工艺。</p><p><img src="https://static.cnbetacdn.com/article/2022/0308/48bc9b748e95f08.png" alt="4.png" referrerpolicy="no-referrer"></p><p>价格方面，预计 RDNA 2 Refresh GPU 不会有太大的变化，但 TGP 功耗可能略有增长，以便同几个月后的问世的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> ARC Alchemist 高端 GPU 一较高下。</p><p>至于 AMD Radeon RX 6950 XT 能否撼动定于 3 月底到来的英伟达 GeForce RTX 3090 Ti 旗舰游戏显卡的王者地位，还请拭目以待。</p>
